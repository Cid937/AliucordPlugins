# Lyrics
Searches lyrics of a song.

## Usage
```
/lyrics [song name]
```

## Example
### Input
```
/lyrics Bury The Light
```

### Output
![image](/screenshots/lyrics_example.png)

## Additional Info
Ported from the [Powercord lyrics plugin](https://github.com/powercord-community/lyrics)
Uses `lyrics-api.powercord.dev`
